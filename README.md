Калькулятор на Python
Простой консольный калькулятор, реализованный с использованием функций.
Функциональность:
· Сложение
· Вычитание
· Умножение
· Деление
· Многократное использование
· Обработка ошибок ввода
Использование:
1. Выберите операцию от 1 до 4
2. Введите два числа
3. Получите результат
4. Для выхода выберите 5
Технические особенности:
· Реализация на чистом Python
· Модульная структура с функциями
· Обработка деления на ноль
· Валидация ввода пользователя
